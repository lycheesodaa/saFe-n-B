**To run the backend:**

1. Set your present working directory as `/backend`.
2. Run `mvn spring-boot:run`


**To run the frontend:**

In a separate terminal, 
1. Set your present working directory as `/frontend`.
2. Run `npm install`.
3. Run `npm run start`
